blitve's training mode

An update to the goeguessr training mode by miraclewhips

⬇️ Install Script

Requires TamperMonkey browser extension

Features

🛰️

Satellite Mode

Switch to satellite view when reviewing round results to get a better sense of the terrain and surroundings.

🖱️

Clickable Coverage

Click on Street View coverage directly from the results map to jump straight to any location.

🏆

Challenges & Duels

Works seamlessly on both challenge and duel result pages.

🔇

Competition Safe

Nothing is shown in-round for timed challenges, so you can safely leave the script running during competitions.

💾

Map Making App Integration

For untimed seeds, save locations to the Map Making App directly in-round without leaving the game.

✨

And more...

How to Install

1
Install the TamperMonkey extension for your browser if you haven't already.
2
Click the Install Script button above. TamperMonkey will open and ask you to confirm the installation.
3
Play a game of GeoGuessr. After each round, the enhanced results view will appear automatically.
