Download CSV of Geoguessr Challenge Results

Export challenge results to CSV with one click

⬇️ Install Script

Requires TamperMonkey browser extension

🖱️

One Click Export

A button appears on any GeoGuessr challenge results page. Click it to instantly download the data.

📄

CSV Format

Downloads a clean CSV with scores, distances, times, and steps for every player and round.

🔒

No Login Required

Uses your existing GeoGuessr session — no extra accounts or API keys needed.

How to Install

1
Install the TamperMonkey extension for your browser if you haven't already.
2
Click the Install Script button above. TamperMonkey will open and ask you to confirm.
3
Navigate to any GeoGuessr challenge results page
4
Click the Download CSV button in the bottom-right corner of the page.

CSV Format

ColumnDescription
NickPlayer username
Total ScoreOverall score in points
Total Distance (m)Total distance error in metres
Total Time (s)Total time taken in seconds
Total StepsTotal steps taken
R1–R5 ScoreScore for each round
R1–R5 Distance (m)Distance error per round
R1–R5 Time (s)Time taken per round
R1–R5 StepsSteps taken per round
